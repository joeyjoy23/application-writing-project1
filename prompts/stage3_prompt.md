# 二、话题词汇锦囊

按**语义场**分类（至少 3 类）。

每类分 **必备级 / 进阶级 / 亮点级**，每层 **5–7 项**。

**每一项格式：英文词块与中文同一行，具体使用例句单独换行。**

```
- **英文词块** / **中文**：… / …
  **具体使用例句**：…（英文一句，贴合本题场景）
```

示例：

- **英文词块** / **中文**：raise concerns about / 对……表示担忧
  **具体使用例句**：I am writing to raise concerns about the noise near our dormitory.

**要求：**
- 必须真实可用、体现本题语境
- 禁止「英文 | 中文 | 例句」单行管道符格式
