# 高考英语应用文 AI 分析系统 — 项目简介

## 这是什么

面向高考英语**应用文**的 AI **备课助手**：固定四阶段工作流，不是自由聊天。教师粘贴一道真题，自动生成从审题到教案的成套材料，可导出 Word、保存历史、断点续跑。

适用：个人备课、教研讨论、课堂示范与讲评整理。

## 项目亮点

- **教研级流水线**：Stage1 输出结构化 JSON 写作蓝图，后续阶段严格据此生成，要点覆盖可校验，避免聊天式「每次不同、易漏要点」。
- **范文有层次、有风格**：PEEL 便签 + 基础版 + 情感共鸣型 / 逻辑思辨型两份高分范文 + 三版对比与升级解析；范文统一 **105–125 词**（忽略原题「约 80 词」等过短要求）。
- **语言与教案一体**：Stage3 功能句型 + 三层话题词汇（必备 / 进阶 / 亮点）；Stage4 按所选 **学生水平** 生成差异化教学路径、易错预警与课后练习。
- **为真实课堂设计**：Prompt 独立在 `prompts/`，教研改模板即可调口径；支持 DeepSeek、智谱、OpenAI、Gemini、百炼、MiMo 等多模型；Stage2/3 并行缩短等待；可选结果缓存降低同题重复成本。
- **可留存、可续用**：同题同模型自动合并历史；每完成一 Stage 即保存，失败也可保留已完成部分；一键导出排版 Word；Cloud 部署可接 Neon 持久化。

## 四阶段一览

**Stage1** 审题（JSON + 教师总结）→ **Stage2** PEEL 与多版范文 → **Stage3** 句型与词汇 → **Stage4** 学情适配教学指南。可全流程，也可单跑某一阶段。

## 快速使用

1. `streamlit run app.py`（或双击 `run.bat`）。
2. 侧边栏选模型、填 Key →「新建」粘贴题目 →「完整流程」或 Stage 1–4。
3. 导出 Word，或在「历史」搜索、载入续跑。
