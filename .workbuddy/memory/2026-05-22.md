## 修复导出 Word 按钮点击后失效的问题

- 问题：`_render_export_buttons` 中 `st.download_button` 的 key 固定为 `export_word` / `export_json`，Streamlit 缓存旧 widget state 后按钮无法再次响应下载
- 修复：用 `question + stage1_summary` 的 MD5 前8位作为 key 动态后缀，内容变化 → key 变化 → 按钮恢复可用
- 修改文件：`app.py` 的 `_render_export_buttons` 函数

## 修复页面刷新闪烁问题（@st.fragment 局部刷新）

- 问题：`advance_run_job()` 每 ~0.2s `st.rerun()` 整页重绘，导致运行中页面持续闪烁
- 修复方案：
  - 新增轮询间隔常量：`_POLL_INTERVAL_FAST=0.5s`（状态转换）、`_POLL_INTERVAL_SLOW=1.0s`（API 等待）
  - 新增 `@st.fragment` 装饰的 `_poll_run_progress()` 函数，用 `st.rerun(scope="fragment")` 实现局部刷新
  - `advance_run_job()` 中线程存活期间改为调用 fragment 轮询，状态转换点仍用整页 rerun
- 修改文件：`ui/run_manager.py`
- 上次尝试一次性 Edit 因 input too long 失败，但确认文件已完整写入，语法验证通过
